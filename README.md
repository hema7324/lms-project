# Library Management System

A simple Command-Line Interface Library Management System.
It uses two MySQL databases to manage users and books/requests.

## Setup Requirements

1. **MySQL Server**
   Ensure you have a MySQL server running locally. Create a user (or use `root`) with privileges to create databases.
   Update the database credentials in `database.py`.

2. **Python Requirements**
   Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the program:
```bash
python main.py
```
Default Admin credentials: Username: `admin`, Password: `admin`.
